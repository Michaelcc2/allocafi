# AllocaFi Marketing Command Center

This folder gives the AllocaFi Digital Team a permanent marketing and advertisement system.

## Purpose

The AllocaFi Top Marketing Agent turns product architecture into clear public messaging:

- What AllocaFi is.
- Who it helps.
- What problems it solves.
- How users can use it.
- Why the non-custodial security model matters.
- Which security benefits can be explained publicly.
- Which technical details must stay internal.

## Folder Contents

1. `00-marketing-agent-charter.md`
   - Agent identity, responsibilities, boundaries, and review partners.

2. `01-brand-message-house.md`
   - Core positioning, tagline options, audience promise, and approved language.

3. `02-use-case-library.md`
   - Real-world user scenarios for personal, business, family, crypto, Legal Core, Tax Ledger, and Vault use.

4. `03-problem-solution-library.md`
   - Market problems, AllocaFi solutions, proof points, and subscriber value.

5. `04-security-messaging-guardrails.md`
   - Public-safe security messaging, restricted details, and claim review rules.

6. `05-advertising-campaign-library.md`
   - Campaign angles, ad hooks, launch copy, short-form scripts, and channel ideas.

7. `06-user-education-and-how-to-use.md`
   - Simple explanations of how to use AllocaFi without overloading new users.

8. `07-review-checklists.md`
   - Marketing, security, compliance, QA, and launch-readiness checklists.

## Activation

Use this folder whenever the user asks for:

- Marketing strategy.
- Ads.
- Landing page copy.
- Product explanation.
- Launch positioning.
- Subscription packaging.
- Security-safe public messaging.
- Customer education.
- Social media scripts.
- Investor or subscriber-facing summaries.

Marketing work must still follow the ADT constitution.
