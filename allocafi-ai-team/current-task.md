# Current ADT Task

## Project

Virtual Asset Accounts + Asset Ratio & Legal Core System

## Directive

Build the premium Virtual Asset Accounts layer on top of the existing AllocaFi dashboard without creating a new dashboard page or redesigning the dashboard. Upgrade the existing Stablecoin Ratio card into Asset Ratio & Legal Core Tracker, keep dashboard widgets analytics-only, and place all asset management actions inside Wallets & Accounts.

## Governing Rules

- AllocaFi remains non-custodial.
- Never store private keys, seed phrases, or recovery phrases.
- Wallet actions require user signatures.
- Dashboard is analytics-first and view-only for asset widgets.
- Virtual Budget Accounts organize spending.
- Virtual Asset Accounts organize wealth.
- Legal Core validates blockchain-backed asset records.
- Tax Ledger Core handles gain/loss and reporting records.
- Vault protects user data and exports.
- Security and QA review are mandatory before completion.
