# AllocaFi Digital Team Prompt Library

This file stores reusable prompts and command patterns for future AllocaFi development sessions.

## ADT Activation Commands

### Command

"Use ADT"

### Definition

Whenever instructed to "Use ADT" or "Use AllocaFi Digital Team", read:

1. `00-adt-constitution.md`
2. `01-adt-agent-directory.md`
3. `02-adt-workflow.md`
4. `current-task.md`

before beginning work.

Then:

1. Review the task through the appropriate agents.
2. Follow AllocaFi governing rules.
3. Preserve architecture.
4. Run security review.
5. Run QA review.
6. Produce implementation plan.
7. Complete task.
8. Produce completion report.
